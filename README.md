🚫 Safe Surf Chrome Extension
Safe Surf is a privacy-focused browser extension designed to help users break free from digital addiction by blocking harmful websites.
It blocks thousands of harmful sites and helps users focus on what truly matters.

🌟 Features
✅ Blocks 50,000+ harmful websites

✅ Simple to install and use

✅ Works offline once installed

✅ Lightweight and efficient

✅ Regularly updatable blocklist (manual updates)

🚀 How to Install
Download the Extension Folder:

Either clone this repo or download the ZIP and extract it.

You can also download it from the official Safe Surf Website (replace this with your site link).

Add to Chrome:

Open Google Chrome.

Go to: chrome://extensions/

Turn on Developer mode (top-right corner).

Click Load unpacked.

Select the extracted folder (SafeSurf-Extension).

✅ Your extension is now ready to use!
